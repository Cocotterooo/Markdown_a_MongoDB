## <img alt="Logo de Anland Supervivencia" src="https://i.imgur.com/fAigIRT.png" width="48px" height="48px"/>  Analand Supervivencia:  

<p align="center">
<img alt="Banner de Analand Supervivencia con un estilo que recuerda al videojuego Minecraft" src="https://i.imgur.com/j8v4euL.png" width="50%" height="100%"/>
</p>


Un proyecto más enfocado a la configuración y diversión ya que está directamente relacionado con el videojuego Minecraft. 

Esto es directamente un servidor dispuesto 24/7 para que cualquier persona del mundo pueda entrar a divertirse. Antes de 2023 este proyecto tenía idea de convertirse en una Network con varias modalidades (Supervivencia, Creativo y Skyblock) pero por dificultad e insuficiencia de capital, se descartó asentándose así en un único servidor Supervivencia con infinidad de detalles.
